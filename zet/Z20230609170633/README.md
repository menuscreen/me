# Ubuntu Server Time Zones

**Check the current timezone**
```bash
timedatectl
```

**List available time zones**
```bash
timedatectl list-timezones | grep America
```

**Set the time zone**
```bash
sudo timedatectl set-timezone America/New_York
```

- Verify the change: `timedatectl`
